# Portfolio Frontend

Minimal React + Vite starter for a personal portfolio website.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build locally.
- `npm run lint` runs ESLint.

## Notes

- The project uses JavaScript with React and Vite.
- The Vite base path is set to `/my_portfolio/` in `vite.config.js` to align with GitHub Pages project-site deployment for the current repository name.
- No portfolio content has been added yet.
